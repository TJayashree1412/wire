As secure as it gets

Wire’s industry-leading security architecture protects all communication — messages, calls, shared documents — with end-to-end encryption. Message encryption keys never leave the device and are changed for each message for maximum security.

Sensitive information can be protected further with timed messages that self-destruct on expiration.

Wire software is regularly audited by independent experts and audit reports are published on our website together with the security whitepaper.

  • End-to-end encrypted

  • 100% open source
